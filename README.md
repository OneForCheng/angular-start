# angular-start

This is an angular project.

## 本地环境

```shell
node -v

npm -v
```
Node >= 6.x
NPM >= 3.10.x

[下载node（npm已包含在node中）](https://nodejs.org/zh-cn/)

## 安装及启动

```shell
```

```shell
npm install -g @angular-cli@1.0.0

npm install

npm start
```

## 构建

### 测试环境

```shell
npm run build-test
```

### 生产环境

```shell
npm run build-prod
```
